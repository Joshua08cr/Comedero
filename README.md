# Comedero
Comedero para gatos automático con arduino


*Descripción*



Pequeño sistema automatizado que dispensa alimento utilizando un <Arduino/ESP32> y un servomotor. 
Detecta la presencia de la mascota mediante un sensor, activa el mecanismo de apertura del contenedor y libera una porción de comida de manera precisa y controlada. 
Ideal para proyectos de domótica y cuidado automático de mascotas.
